# Dockerized Flask Application with Jenkins CI/CD

## 📌 Project Overview

This project demonstrates a complete **CI/CD pipeline using Jenkins and Docker** for a Flask-based URL Shortener application.

The application allows users to convert long URLs into short links and automatically redirects users to the original URL when the shortened link is accessed.

The project showcases modern DevOps practices by automating the build, deployment, and verification process using **Jenkins Pipeline** and **Docker Containers**.

## 🚀 Features

* URL shortening functionality

## ⚙️ Application Workflow

1. User enters a long URL.
2. Flask application generates a unique short code.
3. URL mapping is stored locally.
4. Shortened URL is displayed.
5. Visiting the short URL redirects to the original website.
